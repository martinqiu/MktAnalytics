🚀 Getting Started
  • Download Python
    Visit python.org/downloads and install Python 3.10+. During setup, tick "Add Python to PATH".
  • Download all project files
    Place all 8 files in the same folder on your computer.
  • Run the installer (first time only)
    Double-click install_and_run.bat — it installs all required libraries and launches the dashboard automatically.
  • Open the dashboard in your browser
    Navigate to http://localhost:8501 — Streamlit will usually open this automatically.

⚡ Next time: just double-click run_dashboard.bat — no reinstall needed.

💻 Manual Install (Terminal)
pip install -r requirements.txt
streamlit run fraud_dashboard.py

❓ Troubleshooting
❌ streamlit: command not found
→ Open a terminal and run: pip install streamlit
❌ ModuleNotFoundError
→ Run: pip install -r requirements.txt
❌ Port already in use
→ Run: streamlit run fraud_dashboard.py --server.port 8502
Then open http://localhost:8502
❌ Python not recognized
→ Reinstall Python from python.org/downloads and make sure to check "Add Python to PATH" during setup.
